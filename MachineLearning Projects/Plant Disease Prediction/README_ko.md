# 딥러닝을 이용한 식물 질병 탐지

이미지 기반 식물 질병 탐지는 넓은 밭의 작물을 모니터링하고 식물 잎에 질병 증상이 나타나자마자 탐지하는 데 이점을 줄 수 있으므로 필수적인 연구 주제입니다.

ipynb 파일을 사용하여 모델을 훈련시킨 다음 해당 모델을 사용하여 식물의 질병을 예측할 수 있습니다.
