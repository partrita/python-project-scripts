# 식품 데이터 분석

이것은 인도 음식 요리에 대한 EDA(탐색적 데이터 분석) 프로젝트 보고서입니다.
이 프로젝트는 맛, 주별 소비, 채식/비채식 및 매운맛을 기준으로 다양한 인도 원산지 진미를 분류하기 위해 matplotlib를 사용한 데이터 시각화와 함께 분류를 광범위하게 활용합니다.
